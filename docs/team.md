# 团队分工文档

项目名称：涉水审批智能审核系统  
阶段范围：节点一 - 项目基础架构搭建  
建议团队人数：4 人以内

## 成员与职责建议

### 1. Java 后端负责人

- 负责 Spring Boot 项目搭建
- 负责申请接口、文件上传、CORS 配置
- 负责 JPA 实体、Repository、Service 编写
- 负责预留 Python AI 服务调用接口

### 2. 前端负责人

- 负责 Vue 3 项目初始化
- 负责申请列表、新建申请、初审结果页面
- 负责 Vue Router、Axios、表单与上传交互
- 负责页面样式统一与联调

### 3. Python AI 服务负责人

- 负责 FastAPI 项目初始化
- 负责 `/health` 与 `/review/mock` 占位接口
- 负责后续 LangChain、ChromaDB、MCP 接入预留
- 负责后续知识工具接口设计

### 4. 数据库与文档负责人

- 负责数据库表设计与初始化 SQL
- 负责 SQLite / MySQL 配置说明
- 负责 README、部署说明、提交规范整理
- 负责课程答辩材料与接口说明汇总

## 协作建议

1. 先统一字段命名，再分模块开发。
2. 前后端联调时优先固定接口路径和 JSON 结构。
3. Python 服务当前只返回模拟数据，避免提前耦合真实 AI 逻辑。
4. 每人至少保留自己模块的独立提交记录，方便课程检查。
